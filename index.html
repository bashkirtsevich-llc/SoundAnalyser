Sound Analyser — Программа построения спектрограмм аудиофайлов

Год/Дата Выпуска: 24.02.2017

Текущая версия: 1.4.1

Разрядность: 32bit, 64bit

Язык интерфейса: Русский

Системные требования: Windows 7, 8, 10

Описание: Sound Analyzer может быть использован для анализа аудио файлов, показывая их спектрограммы. Данное программное обеспечение разработано эксклюзивно для товарищей с форума rutracker.org.

  • Поддерживает многие популярные lossy и lossless аудио форматы, за счет использования библиотек FFmpeg.
  • Отображает имя кодека и параметры аудиозаписи.
  • Предоставляет возможность сохранять спектрограммы в PNG формате.
  • Работа с программой может осуществляться с помощью механизма Drag-and-drop.
  • Интуитивно понятный интерфейс.
  • Программа не требует установки.
